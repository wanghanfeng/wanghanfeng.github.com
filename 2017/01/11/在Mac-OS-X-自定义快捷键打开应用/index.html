在Mac OS X 自定义快捷键打开应用

Jan 11, 2017 | Mac操作 | Hits
文章目录

不得不承认，有时候键盘的效率确实比鼠标高的多。充分的利用键盘有助于我们更好的提高工作效率，下面我就来演示在mac下自定义快捷键打开应用：

mac上有一个强大但总被大多数人忽略的一个原生应用Automator,我们可以使用它来实现自定义快捷点打开应用，当然它的强大当然不仅仅局限于此，在这里我就不过多的探讨了，大家自行百度 ：）

先打开Automator

在接下来的界面中选择 服务

之后在菜单框中找到开启应用程序

在这个界面中选择我们想使用快捷键打开的应用，这里拿Sublime来举例

之后按下command + S 保存，给这项服务取个名字，比如说叫打开Sublime，这步我没有截图。

接下来打开系统偏好设置 ，进入键盘设置

找到刚才保存的服务名称，对它设置一个快捷键，建议最好使用三个快捷键，避免与系统或一些软件的快捷键发生冲突。

接下来就可以使用自定义的快捷键来打开应用了！
